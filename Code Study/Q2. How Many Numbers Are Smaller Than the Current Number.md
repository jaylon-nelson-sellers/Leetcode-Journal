# Question
Given the array `nums`, for each `nums[i]` find out how many numbers in the array are smaller than it. That is, for each `nums[i]` you have to count the number of valid `j's` such that `j != i` **and** `nums[j] < nums[i]`.

Return the answer in an array.

**Example 1:**

**Input:** nums = [8,1,2,2,3]
**Output:** [4,0,1,1,3]
**Explanation:** 
For nums[0]=8 there exist four smaller numbers than it (1, 2, 2 and 3). 
For nums[1]=1 does not exist any smaller number than it.
For nums[2]=2 there exist one smaller number than it (1). 
For nums[3]=2 there exist one smaller number than it (1). 
For nums[4]=3 there exist three smaller numbers than it (1, 2 and 2).

**Example 2:**

**Input:** nums = [6,5,4,8]
**Output:** [2,1,0,3]

**Example 3:**

**Input:** nums = [7,7,7,7]
**Output:** [0,0,0,0]

**Constraints:**

- `2 <= nums.length <= 500`
- `0 <= nums[i] <= 100`

# Explanation 

**2 possible solutions:**

1. For every i element, iterate the list with j as index. If i!=j and nums[j]<nums[i], we can update the count for that number. Time: O(n2). Space: O(1)
2. This is optimal solution. We can sort the list and store in another temp list. We use dictionary to store the counts and later use it to create result. Time: O(n log n) bcoz of sort. Space: O(n) bcoz of dictionary.

**Explanation of solution 2:**  
nums = 8, 1, 2, 2, 3

1. We sort the list bcoz we don't want to keep traversing the list back and forth. After sorting, we get temp = 1, 2, 2, 3, 8
2. We traverse temp list. If we find unique element, we know all the previous elements have been recorded. So we add this unique element to the dictionary with index i as value.  
    If duplicate element is encountered, we don't add it and don't update the value in dictionary.
3. After the dictionary is constructed, we create output after doing O(1) lookup in dictionary for nums[i]
# Solution
```python
class Solution:
    def smallerNumbersThanCurrent(self, nums: List[int]) -> List[int]:
        """
        The Intuition (Why Index = Count)
        Imagine a lineup of people sorted by height from shortest to tallest:

        The person at index 0 has 0 people shorter than them.

        The person at index 1 has 1 person shorter than them.

        The person at index 3 has 3 people shorter than them.
                """
        #sort the nums
        temp = sorted(nums)
        #create hashmap and returnArr
        mapping = {}
        result = []

        #iterate through the length of the Arr
        for i in range(len(temp)):
            #if the currVal is not in the hashmap
            if temp[i] not in mapping:
                #okay check if my understanding is correct:
                # add to the map at the index of the current temp value? of i?
                """
                THUS: In a sorted array, the index of a number is exactly equal to                 how many numbers are smaller than it.
                """
                mapping[temp[i]] = i

        # lastly iterate through the nums (no the temp)
        #then append to result with the map
        for i in range(len(nums)):
            result.append(mapping[nums[i]])
        return result
```
