- Solved Easy
```
class Solution:

    def containsDuplicate(self, nums: List[int]) -> bool:

        sets = {}

        for n in nums:

            if n in sets:

                return True

            else:

                sets[n] = True

        return False
```


# Problem 9
- [[485. Max Consecutive Ones]]
- Solved Easy
```
class Solution:
    def findMaxConsecutiveOnes(self, nums: List[int]) -> int:
        maxCons = -1
        temp = 0
        for n in nums:
            if n:
                temp += 1
            else:
                if maxCons < temp:
                    maxCons = temp
                temp = 0
        if maxCons < temp:
                    maxCons = temp
        return maxCons

```

# Problem 10
- [[1295. Find Numbers with Even Number of Digits]]
[Find Numbers with Even Number of Digits - LeetCode](https://leetcode.com/problems/find-numbers-with-even-number-of-digits/description/)
- Solved O(n)
```
class Solution:
    def findNumbers(self, nums: List[int]) -> int:
        count = 0
        for n in nums:
            temp = str(n)
            if not len(temp) % 2:
                count += 1
        return count
```
